Accessible SSL Warning Design

HCI Usable Security 2019

A pilot study conducted by a team six to identify issues with current free screen readers when confronted with SSL readings. We developed a chrome extension that would produce a new multi-modal warning instead of Chrome's 2019 SSL warning design.
